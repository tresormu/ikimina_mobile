# IkiminaPass Mobile

Expo React Native starter inspired by the live IkiminaPass web experience at `https://ikimina-web.vercel.app/`.

## What is included

- Expo SDK 54 baseline
- TypeScript setup
- Native landing experience for:
  - Home
  - How It Works
  - Services
  - Pricing
  - Contact
- Reusable content file in `src/content.ts`

## Run locally

```bash
npm install
npm start
```

Then scan the Expo QR code with Expo Go.
